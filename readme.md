# 🍺 Golden Draught Brewery Search

Golden Draught is a premium brewery discovery tool that combines real-time API data with AI-powered enrichment. Instead of just providing an address, it uses Google Search and Google Gemini AI to curate professional descriptions, food availability, and specific beer statistics for breweries in any given city.

## 🚀 Features
- **Location Search:** Finds breweries via the Open Brewery DB.
- **AI Enrichment:** Uses **Gemini 3.1 Flash Lite** to summarize brewery vibes and food options.
- **Web Intelligence:** Integrated **Google Custom Search API** to fetch real-time web data for more accurate beer lists (ABV/IBU).
- **Premium UI:** A sleek, dark-themed responsive interface.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **AI:** Google Generative AI (Gemini)
- **APIs:** Open Brewery DB, Google Custom Search
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
